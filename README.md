# Stock Price Prediction System

This project is a web-based stock prediction system developed using Java and Spring Boot.
It uses machine learning models and divergence-based trading strategies to analyze and predict
price movements of selected NIFTY 50 stocks.

Technologies Used:
- Java (Spring Boot)
- HTML, CSS, JavaScript
- MySQL
- Hibernate
- SMILE Machine Learning Library

Features:
- Historical data analysis
- RSI divergence-based strategy
- Machine learning prediction models
- Backtesting with MAE, RMSE, MAPE
- Graphical visualization of results

Note:
This project is developed for academic purposes.
